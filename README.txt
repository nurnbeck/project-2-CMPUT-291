Usage:
-------------------------------------------------
To run this program, type ./mydbtest <argv> where argv is one of the following:
btree
-btree
-b
hash
-hash
-h
indexfile
-indexfile
-i
where "btree", "-btree", "-b" for btree,
      "hash", "-hash", "-h" for hash,
      "indexfile", "-indexfile", "-i" for indexfile.

IMPORTANT:
If permission denied, then try the following:
Give shell permissions to execute script. This is done with chmod
  command in command shell, for example, use 
	chmod 755 mydbtest
  to give persmissions to read, write, and execute permission.
